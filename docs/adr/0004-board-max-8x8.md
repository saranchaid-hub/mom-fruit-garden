# กระดานสูงสุด 8×8 (เล็กกว่าต้นฉบับ) เพื่อขนาดเม็ดที่ผู้สูงอายุกดได้

Candy Crush ใช้กระดานถึง 9×9 แต่บนจอมือถือ ~375px เก้าคอลัมน์ทำให้เม็ดเล็กกว่า 40px — เล็กเกินไปสำหรับนิ้วและสายตาผู้สูงอายุ เราจำกัดกระดานที่ 8×8 เพื่อให้เม็ดมีขนาด ≥45px เสมอ ด่านแรกๆ ใช้ 6×6-7×7 ความหลากหลายของด่านมาจากรูปทรงกระดาน (ช่องปิด) เป้าหมาย และจำนวนชนิดผลไม้แทนขนาด

ประกอบกับข้อกำหนด accessibility อื่นที่ยึดหลักเดียวกัน: ผลไม้ทุกชนิดต้องแยกได้ด้วยรูปทรง (ทดสอบด้วยสกรีนช็อต grayscale), ฟอนต์ไทย ≥20px, ปุ่ม ≥48px — เมื่อความสวยงามหรือความจุด่านชนกับการมองเห็น/การกดของคุณแม่ ให้อย่างหลังชนะเสมอ
