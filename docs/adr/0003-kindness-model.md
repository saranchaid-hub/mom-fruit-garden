# Kindness Model: คง move limit ไว้ แต่การันตีว่าด่านไม่มีทางตัน

"ตัดความกดดันออก" ตีความได้สองทาง: ตัดการแพ้ทิ้งทั้งหมด (ไม่มี move limit) หรือคงความท้าทายไว้แต่ถอนพิษของมัน เราเลือกอย่างหลัง — ความภูมิใจตอนผ่านด่านคือความเพลินหลักของเกมแนวนี้ ถ้าทุกด่านผ่านเสมอเกมจะจืด

ดีไซน์ที่เลือก: move limit เผื่อเยอะกว่าเกมทั่วไป ~35-50%, ไม่มีระบบชีวิต/รอเวลา/โฆษณา/การซื้อ, ไม่ผ่านด่าน → ข้อความอ่อนโยน + เล่นซ้ำทันที (Gentle Fail), ไม่ผ่านด่านเดิม 2 ครั้งติด → รอบถัดไปแถม +5 ตาอัตโนมัติ (Mercy Moves) ซึ่งสะสมความง่ายขึ้นเรื่อยๆ จนผ่านได้เสมอ

## Consequences

- ค่า moves ของทั้ง 60 ด่านต้องจูนตามสูตรนี้ (ตาที่ผู้เล่นทั่วไปใช้ × 1.35-1.5) — เปลี่ยนโมเดลทีหลังคือรีบาลานซ์ทั้งเกม จึงบันทึกเป็น ADR
- ต้องเก็บ failStreak ต่อด่านใน SaveData เพื่อให้ Mercy Moves ทำงานข้ามการปิดเปิดแอป
