# เผยแพร่เป็นเว็บแอป PWA บน GitHub Pages แทน native app

เกมต้องเล่นได้ทั้งคอมและมือถือโดยไม่มีค่าใช้จ่ายใดๆ ตลอดอายุโปรเจกต์ เราเลือกเว็บแอป PWA โฮสต์บน GitHub Pages: โค้ดชุดเดียวเล่นได้ทุกเครื่อง ติดตั้งไอคอนหน้าจอโฮมได้ (Add to Home Screen) และเล่นออฟไลน์ได้ผ่าน service worker — native app ให้ประสบการณ์เนียนกว่าแต่ค่าธรรมเนียมนักพัฒนา Apple (~$99/ปี) ขัดเงื่อนไข "ฟรี" โดยตรง

## Consequences

- ความคืบหน้าเซฟลง localStorage แยกตามเครื่อง — เล่นบนคอมกับมือถือความคืบหน้าไม่ sync กัน (ยอมรับ: ไม่มีอะไรเสียหาย ทุกด่านย้อนเล่นได้เสมอ)
- URL ของเกมผูกกับชื่อ repo (`base` ใน vite.config.ts ต้องตรงกับชื่อ repo บน GitHub)
